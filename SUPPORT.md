# Support QuickChat

QuickChat is an open-source project and will always remain free to use. Maintaining an open-source project requires time spent on development, bug fixes, testing, documentation, maintenance, and community support. 

If QuickChat has been useful to you or your team and you'd like to support its continued development, you are welcome to contribute in various ways.

## Ways to Support

### 1. ⭐ Star the Repository
The easiest way to support QuickChat is by starring the repository on GitHub. This increases visibility and helps other developers discover the tool.

### 2. 🐛 Report Bugs
If you encounter any issues, reporting them helps improve the stability of the software. Please use the GitHub Issues tracker to report bugs.

### 3. 💡 Suggest Features
We are always looking to improve. If you have ideas for new features or enhancements, feel free to open a feature request on GitHub.

### 4. 🔧 Contribute Code
Pull requests are highly encouraged! Whether it's a minor bug fix, a performance improvement, or a major new feature, your code contributions are welcome. Please see our [CONTRIBUTING.md](CONTRIBUTING.md) for guidelines.

### 5. 📖 Improve Documentation
Great documentation is vital. If you find typos, unclear explanations, or areas lacking detail, contributing to the documentation is incredibly valuable.

### 6. ❤️ Support Development Financially (Optional)

If QuickChat has been useful to you or your team, and you'd like to support its continued development financially, you can contribute using UPI. Your support helps improve QuickChat, maintain the project, fix bugs, and build new open-source features.

Please reach out to support@quickchat.io for business inquiries.

## Thank You

Thank you for being part of the QuickChat community. Your usage, feedback, and contributions are what make this project possible!
